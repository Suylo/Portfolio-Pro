AP N°4 - GSB Médecins

Ce projet fera partie d'un bouquet de quatre projets réalisés par l'étudiant au cours de ses deux années de BTS. Le candidat devra présenter deux de ses projets dans le cadre de l'examen oral qui aura lieu en fin de deuxième année.

Technologies utilisées :
Pour la réalisation de ce projet, j’ai utilisé le langage de Programmation Orienté Objet Java. J’ai pu réaliser le logiciel avec la bibliothèque JavaFX.

AP N°3 - GSB Visite

GSB Gestion de Visites, qui est la seconde variante des projets GSB proposés, est un sujet qui doit être réalisé dans le cadre du BTS SIO. Ce projet fera partie d'un bouquet de quatre projets réalisés par l'étudiant au cours de ses deux années de BTS. Le candidat devra présenter deux de ses projets dans le cadre de l'examen oral qui aura lieu en fin de deuxième année.

Application à réaliser :
L’entreprise envisage de permettre aux visiteurs de gérer leurs visites à travers une application. Les visiteurs pourront gérer leurs rapports (ajouter, modifier, rechercher) et gérer leurs médecins (voir les rapports du médecin, modifier, rechercher). Cette application devra utiliser un langage de programmation orienté objet ainsi que l’architecture Modèle-Vue-Contrôleur

Technologies utilisées :
Pour la réalisation de ce projet, j’ai utilisé PHP Orienté Objet avec le Framework Laravel, ce dernier respectant l’architecture Modèle-Vue-Contrôleur. Pour la partie visuelle, je me suis orienté vers l'utilisation du framework Bulma. J’ai utilisé Laravel afin de sortir de ma zone de confort et me confronter à un environnement de développement qui est présent partout, dans toutes les entreprises : l’utilisation de frameworks. Ces derniers sont utilisés afin d’augmenter l’efficacité des développeurs en réduisant leurs charges de travails (automatisation des tâches, sécurité, ...).

AP N°2 - Sharing-Picture

SharingPictures est le premier de projet de la première année du BTS. J'ai choisi comme nom "Sharing-Pictures" car sur ce site il est possible de partager ses photos !
Objectifs :
- Réaliser un site web MVC (Modèles Vues Contrôleurs) en utilisant la Programmation Orientée Objet en PHP.
- Réaliser des opérations avec la base de données : Affichage, Ajout, Modification, Suppression (CRUD).
- Le choix du sujet était libre. J'ai choisi un site web permettant de partager ses photos à la Instagram-like. Il est possible de commenter / aimer / ajouter / supprimer / modifier des photos !
